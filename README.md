### Cartiq-Power-BI-Advanced-Project

# 📊 Cartiq – Advanced E-Commerce Insights Dashboard

## 🔍 What?

Cartiq is an advanced Power BI dashboard that analyzes e-commerce data to track sales, profit, and customer behavior with dynamic KPIs and interactive visuals.

---

## 🎯 Why?

The purpose of this project is to:

* Track monthly business performance (MoM growth)
* Identify top-performing categories and regions
* Analyze customer segments and returns
* Enable data-driven decision making

---

## 🛠️ How?

### Tools & Technologies used:

* **Power BI** → Dashboard & visualization
* **DAX** → Advanced measures (MoM, Previous Month, Dynamic Metrics)
* **Excel** → Data cleaning & preprocessing
* **Data Modeling** → Star schema (Fact & Dimension tables)

---

## 🚀 Features (Advanced 🔥)

### 📌 Dynamic KPI Cards

* Monthly Sales, Profit, Units
* Previous Month Comparison
* MoM Growth % with ↑ ↓ indicators
* Conditional formatting (Green/Red)

### 📊 Interactive Visuals

* Sales by Category, Region, SubCategory
* Dynamic Line Chart (Year ↔ Month view)
* Donut charts with parameter-based switching

### 🎛️ User Controls

* Year & Month slicers
* Category & SubCategory filters
* Field Parameters for dynamic analysis

### 🎨 Professional UI Design

* Custom KPI cards (not default visuals)
* Icons, shadows, spacing, clean layout
* Dashboard navigation using bookmarks
